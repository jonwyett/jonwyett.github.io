FPC Utilities

Credential Cropper

This tool helps you crop photos to the correct aspect ratio for FPC credentials. It supports a variety of file types, including HEIC (iPhone format), and can also repair many issues with corrupt images. If FoJEMS doesn't accept your original file, try processing it with this utility first.

iPhone Image Converter

Effortlessly convert HEIC files from an iPhone into JPGs. This tool supports batch processing, allowing you to convert multiple files simultaneously for added convenience.

Transcript Cleaner

Clean up your transcripts by removing time-codes. Currently, this tool is optimized for use with transcripts generated in Slack.
